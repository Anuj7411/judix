04 / WHERE YOU USE IT

Two front doors, one engine.

Wire Judix in once. Every turn and every answer is scored live, automatically. Your users write nothing.

IN YOUR APP

An HTTP call your agent makes.

Wire Judix into any agent you build, by hand, with LangChain or CrewAI, or one you wrote with Codex or Claude Code. One line, and every turn is scored automatically.

You see it where your agent runs: live in your terminal as it streams, or in your own UI if you build one.

05 / THE FLYWHEEL

Every failure it catches becomes a test.

Judix scores every turn, so it knows exactly which runs failed. It turns each one into a regression test automatically. Your eval suite writes itself, from real production failures, for both agents and RAG.

✓Run all · export JSON

See every claim, grounded or contradicted.

Judix decomposes a RAG answer into atomic claims and grounds each against the sources. Averages would hide the one bad claim, so Judix caps the score when any claim contradicts.
